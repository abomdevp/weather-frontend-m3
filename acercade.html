Acerca de este proyecto

Descripción

Este es un proyecto mínimo viable (MVP) desarrollado como práctica de frontend. Muestra el estado del clima en diversas localidades y permite acceder al detalle con un pronóstico semanal. Fue construido para ejercitar HTML5 semántico, Bootstrap para diseño responsivo y JavaScript para la interacción.

Tecnologías utilizadas

  • HTML5 semántico
  • Bootstrap 5 (Grid, Cards, Navbar, utilidades)
  • JavaScript
  • Mock data para simular información de clima

Objetivo del proyecto

El propósito principal fue practicar la estructura de una aplicación web, el diseño adaptable y la interacción básica con JavaScript. Es un punto de inicio para futuros desarrollos y mejoras.

Proyecto de Francisco González - MVP usando HTML, Bootstrap y JavaScript
